축적된 노하우와 안정적인 기술력으로 기업의 성장과 디지털 혁신을 함께 만들어갑니다.
경험 기반의 전문 컨설팅과 유연한 시스템 커스터마이징을 통해
고객 맞춤형 IT 서비스를 제공합니다.

회사소개

KYS-System은 20년 이상의 ERP 개발 및 유지보수 경험을 바탕으로, 다양한 산업에 최적화된 IT 솔루션을 제공하는 전문 개발 업체입니다.

주요 개발 정보

  • 개발 기간: 2013년 12월 ~ 2019년 1월 (약 5년)
  • 개발 모듈: 인사, 영업, 생산, 구매, 품질, 재고관리
  • 총 개발 화면 수: 135 화면
  • 개발 언어: C#.NET, WPF, 웹서비스
  • 개발툴: Visual Studio 2017 → 2019 (2019년 5월 25일 기준)
  • 데이터베이스: MS-SQL SERVER 2017 EXPRESS (무료, 업그레이드 가능)

운영 현황

  • 현재 실 운영 중인 ERP 시스템
  • 1 ~ 20대 PC까지 사용 가능한 네트워크형 프로그램

웹 기능

  • 공지사항 및 생산현황 확인 기능 포함 (2019년 6월 16일 추가)
  • 웹 언어: ASP.NET MVC Core, 서버: Windows IIS
  • 추가 용도에 맞춘 기능 확장 가능

가격 안내

가격 정책은 별도 문의 부탁드립니다.

  • (주)한동허브 (약제 제조공장): MES 생산관리 및 수불관리 – 2019년 11월
